# FurnitureCo – Furniture E-commerce Website

FurnitureCo is a responsive frontend furniture store website built using HTML, CSS, and JavaScript.

## Features
- Modern furniture product listing
- Category browsing
- Product detail pages
- Login / Signup UI
- Add to cart interaction
- Responsive layout

## Technologies Used
- HTML
- CSS
- JavaScript
